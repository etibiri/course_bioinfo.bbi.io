Introduction la Programmation en Bioinformatique

La programmation est devenue essentielle pour analyser et grer les donnes biologiques. Ce cours explique comment installer un environnement de dveloppement adapt.

1. Installer Ubuntu 22.04 LTS sous Windows 10+

1.1. Installer Bash pour Windows

Pour installer Bash pour Windows, suivez les tutoriels ci-dessous :

  • Installer le shell Bash Linux sous Windows 10 avec WSL (YouTube, 2020)
  • Everything You Can Do With Windows 10s New Bash Shell (HowToGeek, 2018)

Ensuite, ouvrez Bash et excutez les commandes suivantes :

sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip python3-dev python3-setuptools
sudo apt install git

1.2. Installer Jupyter Notebook

Excutez la commande suivante pour installer Jupyter Notebook :

sudo pip3 install jupyter

1.3. Vrification de l'installation

Pour vrifier que Jupyter Notebook a t install avec succs, lancez cette commande :

jupyter notebook

1.4. Installer le kernel Bash

Pour ajouter le kernel Bash, excutez :

pip install bash_kernel
python3 -m pip install --upgrade pip
python3 -m pip install bash_kernel
jupyter notebook
